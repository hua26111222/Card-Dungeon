# Card Dungeon — 项目进度

## 已完成功能

### 核心系统
- 卡牌数据库（DB）：40+张牌，含攻击/防御/技能/诅咒
- 回合制战斗：AP、格挡、中毒、眩晕、缴械、吸血等状态
- 地图系统：随机生成、多事件节点
- 存档/读档：localStorage 自动存档
- 章节系统：3章 + Boss + 祝福/修改器

### 战斗画面
- KOF风格战斗特效：敌人受击抖动、格挡弹出、中心战斗文字
- 敌人对话气泡：嘲讽/受伤/低血/庆祝台词（按trait分类）
- **SVG人形立绘系统**（2026-06-29新增）：
  - 9种trait外观映射（骑士/魅魔/骷髅/暴君/术士/流浪者/盗贼/施法者/吸血鬼）
  - 5种武器类型（剑/斧/法杖/匕首/锤）
  - 8种头盔/发型类型（全盔/开盔/兜帽/法师帽/角/王冠/短发/长发）
  - 参数化颜色调色板，每类trait固定配色
  - 玩家每局随机外观（基于随机seed）

### 商店与经济
- 黑市：4格卡牌，35%骗局概率，虚假描述
- 锻造炉：带升级前/后属性对比面板
- 祝福预览：进章节前可看到祝福内容

### 敌人系统
- 12种trait + 详细性格描述/口头禅
- 多种AI类型（power/skill/aggro）
- 精英/Boss等级差异

### 动态故事
- 章节开场白：即时生成，每次不同
- 按层数分3段叙事（深渊召唤/中段成长/最终决战）

## 宿命感系统（2026-06-29）
- **全层预演算**：进入新章节时，整层地图（含所有战斗节点的敌人）一次性生成，宿命已定
- **楼层简介**：进入章节前显示楼层气氛描述（8种气氛：阴险/毒雾/财富/黑暗/强者/古老/诅咒/猛兽）
- **宝箱节点**：新增 💎 宝箱地图节点（~9%概率），内含3张强力卡牌，选择1张带走

## 战斗平衡与体验修复（2026-06-29）
- **能量/抽牌奖励收紧**：除每层Boss的奖励外，其余途径（小怪奖励、祭坛、黑市等）不再提供"+1行动点上限"或"+1抽牌数"，避免战斗节奏失衡
- **手牌上限10张**：无论何种加成都不能超过10张手牌，防止抽牌堆为空导致无法继续抽牌
- **赌馆"全或无"结算画面**：赢得赌局后会展示获得的具体卡牌（之前是静默加入牌组，看不到结果）
- **战利品贴合敌人特征**：战后奖励的3张卡牌从刚击败的敌人所属流派（如吸血鬼出吸血/治疗牌，暴君出力量/重击牌）中抽取，且展示顺序随机打乱，不再按价值排序
- **战前强度评估**：每层开局会预先生成全部敌人，但战斗开始前会比对当前玩家强度，若敌人明显偏弱会自动增强其血量与攻击力，避免"宿命已定"的预生成敌人显著落后于玩家成长

## 无限地牢 + 探险排行榜（2026-06-30）
- **移除通关上限**：原本第3层为最终层、击败Boss后弹出"胜利"画面并结束游戏；现已移除，Boss战胜后永远进入下一层，层数无限递增，难度随层数持续提高
- **死亡即排行**：玩家死亡时自动把本局战绩（到达层数、造成总伤害、击杀数、称号）存入本地排行榜（最多保留50条历史记录）
- **排行榜界面**：可在主菜单和死亡画面查看，支持按"层数/伤害/击杀"三种方式切换排序，显示前20名
- 原"胜利画面"相关代码（showVictory等）已整体替换为排行榜功能，不再有"通关"概念——游戏现在是纯粹的"能走多远"挑战

## 章节开场简化（2026-06-30）
- 进入新层时不再重复展示"开局随机卡牌"和"完整牌组"列表，合并为一个"查看牌组"按钮
- 章节开场文字改为按层数衔接的连续剧情（不再是孤立的"第N章"标题），强化"地牢逐层下潜"的叙事感
