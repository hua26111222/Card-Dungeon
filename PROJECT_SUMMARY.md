# Card Dungeon — 项目进度

## 敌人生成系统体检与优化（2026-07-01）
对"敌人怎么生成、怎么变强、怎么出招"这套系统做了一次全面检查，处理了4项代码整理 + 1项数值调整：
- **难度计算公式去重**：敌人强度随楼层/地图格子等级增长的公式，原本在5个地方各自重复抄写（生成敌人、地图悬浮预览、战前情报页、战前补强检查），现已统一收进一个 `calcTierScale()` 函数，以后只需要改一处，不会再出现"地图上看到的预估强度"和"真正打起来"对不上的风险
- **清理12种已经不会出现的旧敌人模板**：地图现在都是用"战斗力/卡牌破坏/特性"三维组合的新方式预先生成敌人，旧版固定的12种小怪/精英模板早已经抽不到了，予以删除，只保留仍在使用的5个Boss模板
- **补全"aggro"敌人风格，不再是摆设**：敌人原本设计了power/skill/aggro三种出招风格，但aggro从未真正实现、也没有敌人用到。现在aggro被实现为"几乎不管不顾、每回合都抢着进攻"的莽夫风格，并把"掠夺者""暴风掠夺者""影子双子""深渊触手怪""渴血狼人""双刃剑客""雪原狼王"这7种以连续多次攻击为招牌动作的敌人特性，从原本的power改为aggro，让它们的出招手感更贴近"疯狂进攻"的设定
- **删除战前情报页里一段永远不会显示的敌人预览卡片代码**：该卡片依赖的变量在全部代码里从未被真正赋值，属于早期版本遗留的死代码，确认删除不影响任何现有功能
- **取消敌人攻击的伤害安全阀（数值调整，非bug修复）**：原本每种敌人的攻击力都被限制在"玩家最大生命值的一定比例"以内，作为防止数值算错后一拳秒人的保险丝。应用户要求直接取消这道上限，敌人的招式伤害现在完全由其自身倍率决定，不再被人为封顶——尤其是"暴君""嗜魂怪"残血斩杀这类原本会被砍掉一部分的爆发技，现在会打出完整倍率的伤害，游戏会比之前更硬核、更容易暴毙，请留意后续实际游玩的难度手感

## 体验修复 + 战场日志 + 敌人特性扩容 + 称号系统（2026-06-30）
本轮共12项任务，按顺序逐一修复体验问题、补全机制，并大幅扩充敌人内容：
- **地图节点不再泄密**：地图上的战斗节点按钮文字和卡牌配色之前会间接暴露"这是精英还是普通怪"，已统一改为不泄露具体类型的呈现方式
- **修复选2张牌只能选中1张的bug**：多选逻辑里状态更新时机不对，导致第二次点击经常没生效，现已修复为稳定多选
- **状态栏新增中毒显示**：玩家中毒时，状态栏会直接显示当前中毒层数，不用再靠猜
- **战斗文字降噪**：连击、格挡等动作如果没有额外特殊效果（比如没有触发吸血、没有破甲等），不再弹出战斗大字提示，减少无意义的视觉打断
- **非boss战斗不再直接加属性**：小怪战斗的奖励只给卡牌/金币，"直接提升攻击力/生命上限"这类强力奖励收回给boss专属，避免数值膨胀过快
- **小偷类敌人必定逃跑**：原本"血量低/拖太久就跑"的设定经常因为时机判断问题不生效，现在血量低于阈值或回合数超限后会100%触发逃跑
- **修复"战斗获得战利品"事件不触发真实战斗的bug**：原代码只是直接给奖励，没有真的进入战斗流程，现已修复为先打一场仗，赢了才给战利品
- **玩家初始血量下调**：原先玩家初始血量远高于普通敌人，前期几乎不会有压迫感，现已调整到与普通敌人接近的量级
- **新增战斗日志**：战斗界面新增"📜 战斗日志"入口，记录本场战斗里敌我双方每一次出招和效果（伤害、格挡、异常状态等），关闭/进入其他界面会自动收起，不会挡在屏幕上
- **卡牌"稀有度"改为数值打分**：原本卡牌上显示"普通/罕见/稀有"文字分级，容易让人误以为是固定品质标签；现在统一显示一个综合实力评分（如"✦42"），分数由卡牌自身的攻击/防御/治疗/特殊效果等综合计算得出，黑市开出的牌也同步改为打分制
- **敌人特性从10种扩充到50种**：新增40种全新敌人特性（骷髅兵、火焰术士、冰霜巫婆、蛛后、食人花、石像鬼、暗影刺客、雷霆战士等），每种都有独立的招式组合、外观配色、登场/受伤台词和专属战利品卡池；新增特性全部基于已有的18种"动作原语"（攻击/格挡/中毒/眩晕/缴械/偷牌/封卡等）组合而成，无需改动战斗引擎本身。地图随机生成敌人时已自动从全部50种特性中抽取，新增内容立刻可在正常游戏中遇到，且敌人立绘会基于特性名自动生成一套独特配色，无需额外画图
- **新增称号系统**：游戏内本就有"根据卡组构成生成称号"的逻辑（如"嗜血的残忍斗士""坚不可摧的铁壁守护"），本轮在此基础上补充了"专门猎杀某类敌人"的称号判定（击杀同一特性敌人达4只以上会获得"XX的克星"称号），并让称号在每次进入新的一层时自动重新计算——如果称号相比上一层发生了变化，进入新章节的界面会用一个发光的"称号已晋升"横幅提示玩家

- **能量/抽牌奖励收紧**：除每层Boss的奖励外，其余途径（小怪奖励、祭坛、黑市等）不再提供"+1行动点上限"或"+1抽牌数"，避免战斗节奏失衡
- **手牌上限10张**：无论何种加成都不能超过10张手牌，防止抽牌堆为空导致无法继续抽牌
- **赌馆"全或无"结算画面**：赢得赌局后会展示获得的具体卡牌（之前是静默加入牌组，看不到结果）
- **战利品贴合敌人特征**：战后奖励的3张卡牌从刚击败的敌人所属流派（如吸血鬼出吸血/治疗牌，暴君出力量/重击牌）中抽取，且展示顺序随机打乱，不再按价值排序
- **战前强度评估**：每层开局会预先生成全部敌人，但战斗开始前会比对当前玩家强度，若敌人明显偏弱会自动增强其血量与攻击力，避免"宿命已定"的预生成敌人显著落后于玩家成长

## 无限地牢 + 探险排行榜（2026-06-30）

- **排行榜界面**：可在主菜单和死亡画面查看，支持按"层数/伤害/击杀"三种方式切换排序，显示前20名
- 原"胜利画面"相关代码（showVictory等）已整体替换为排行榜功能，不再有"通关"概念——游戏现在是纯粹的"能走多远"挑战

## 概率化稀有度 + 敌人特性强化 + 祭坛驱散诅咒（2026-06-25）
- **地图稀有度改为百分比**：节点角标从文字（普通/罕见/稀有/史诗/传说）改为遭遇概率百分比（45%/28%/15%/8%/4%），避免"稀有度"文字被误解为牌的品质；相关5档定义提取为全局常量 `VAL_TIERS`/`CP_VAL_IDX`/`TYPE_VAL_IDX`，地图角标、悬浮提示、奖励生成统一引用，不再各自维护一份
- **眩晕/缴械视觉反馈修复**：之前被眩晕/缴械后手牌看起来和平时一模一样，只有点击时才弹提示；现在手牌会直接变灰不可点（复用现有 `.unp` 样式），意图栏也补充了"下回合你将无法出牌/无法使用攻击牌"的说明文字
- **修复"偷牌/摧毁牌"实际无效的bug**：原代码只把卡从战斗中的 `hand`/`drawPile` 移除，但没有从持久牌组 `battle.deck` 里删，导致小偷"偷走逃跑"、强盗"当场砸烂"都是摆设，下一场战斗卡照样还在。现在偷牌/摧毁牌会真正从 `battle.deck` 永久移除；击杀小偷可把战利品（含被偷的牌）找回并重新放回牌组
- **新增"能量封锁"机制**：魅魔（trait）使用新的 `lock_ap` 技能，直接压低玩家的最大行动力上限并持续2-3回合，比旧的 `drain_ap`（只吸一回合、下回合自动满血）更有"极端"的压制感；封锁到期会自动归还，且至少保留1点行动力避免彻底卡死
- **移除地图节点冗余的"精英"文字角标**：百分比角标已经能体现稀有度，原本同时显示的"精英"文字是多余信息，予以删除（战斗内、奖励页等其他位置的"精英"标签不受影响）
- **奖励生成与节点稀有度挂钩，强化"刷卡"体验**：战后奖励的3张卡，会按照该节点稀有度百分比（越稀有的节点，敌人随身卡池里出的卡越多、整体也越偏稀有）来决定构成，让"专门去打某类敌人来刷想要的卡"这件事变得可行；同时修复了一个真实存在的bug——旧逻辑在特定情况下会一次性给出最多5张卡，而不是设计中的3张
- **祭坛新增"驱散诅咒"功能**：当牌组中存在诅咒牌时，祭坛会出现"驱散诅咒"入口，需要玩家自行选择1张诅咒牌 + 1张稀有/罕见牌作为祭品，再额外献祭20%最大生命值，即可彻底移除该诅咒；完全是可选项，不去祭坛、不诅咒缠身的玩家不会受到任何影响

## 敌人特性进一步极端化（2026-06-30）
在小偷/强盗/魅魔之后，把剩余几种特性也补上了真正"极端"、彼此互不重复的招式，每种特性现在都有独一无二、能改变玩法的标志性机制：
- **吸血鬼**：新增"血脉烙印"——攻击之外还会直接永久夺走玩家一部分最大生命值上限（不是当前血量，战斗结束也不会恢复），转移给自己增加血量上限，真正体现"嗜血"的代价
- **疯狂科学家**：新增"基因污染"——会把玩家手牌/抽牌堆里随机一张好牌原地永久变异成诅咒牌（不是销毁，是污染替换），呼应"疯狂实验"的诡异感
- **暴君**：新增"终焉处决"——平时只是较强的普通攻击，但只要玩家生命低于35%，这一击会变成2.2倍伤害的处决，残血阶段极度危险，逼玩家不能"苟"着拖时间
- **掠夺者**：新增"狂风骤雨连击"——一回合内连续打出3次攻击，区别于其他敌人的单次重击节奏
- **控制师**：在原本的眩晕+缴械基础上追加了能量封锁（复用魅魔的lock_ap机制），凑成眩晕/缴械/能量封锁三重压制，真正做到"彻底动弹不得"
- **游荡者**：用"灵体封印"（封印卡牌本场战斗不可用）替换了原本和小偷重复的偷牌技能，避免特性撞车，更贴合"幽灵吞噬记忆"的气质
- 以上所有新机制都补充了战斗内意图栏的中文提示文字，玩家在敌人出招前就能看到警告（如"残血时极度危险"），不会出现"不知道发生了什么"的情况；地图节点悬浮预览里的特性介绍文字也同步更新，准确反映每种敌人现在真正的招式

## 暗牌×2效果补全（2026-06-30）
此前"暗牌揭晓"（地图上的隐藏节点被翻开后触发，提示效果×2）这个机制只在4个地方真正生效（战斗血量/金币奖励、营火回血、酒馆折扣），其余节点类型完全没有体现，等于"提示了双倍但什么都没双倍"。现已为剩余5类节点全部补上：
- **赌馆**：小赌/豪赌/神赌选中的卡牌会获得双倍数量；全或无赢局时卡牌从2张变4张；进入赌馆和结算画面都会显示"🎴 暗牌效果"提示
- **事件（13种事件全覆盖）**：所有事件里"纯粹的奖励"（金币、HP回复、最大HP提升、直接获得卡牌）按效果翻倍，但代价/惩罚/诅咒数量保持不变（比如献祭HP的成本不会跟着变高，只有换来的奖励会翻倍），逐条核对了13个事件、约18处具体选项；触发稀有卡奖励的事件，稀有卡数量也会从1/2张变成2/4张
- **黑市**：所有商品价格直接打五折（按效果倍数整除），相当于"暗牌"的省钱效果延伸到了黑市这种只卖不送的场景
- **祭坛焚烧**：高价值祭品换来的生命馈赠和卡牌馈赠都翻倍（卡牌变成拿2张同款），中等价值祭品的回血量也翻倍；低价值祭品的惩罚分支不受影响，献祭风险不会因为暗牌而变得更危险
- **宝箱**：原本固定选1张，现在可以选2张带走（卡牌选完会从候选中移除，连续选两次后才会真正退出宝箱画面）
- 所有新增的双倍效果都配有醒目的橙色提示文字，玩家在做选择前就能看到"本次奖励翻倍"，不会出现选了之后才发现有彩蛋的情况
